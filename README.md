
# Bethlehem to Seagrove: The Memoir of Jack C. Lail

**"Quiet, unassuming, and slow to speak but careful to listen."**

This repository contains the source code and content for the official website of *Bethlehem to Seagrove*, a memoir documenting the 93-year journey of Jack C. Lail.

The site serves as a digital companion to the book, which captures nearly a century of historyâ€”from a childhood in rural Bethlehem during the Great Depression to the founding of Mid-State Plastics and a life of civic leadership in Seagrove, North Carolina.

## Project Overview

* **Book Release:** March 2026 (Available on Amazon in Paperback and Kindle)
* **Website:** [bethlehemtoseagrove.com](http://bethlehemtoseagrove.com)
* **Key Themes:** Family genealogy, Southern history, entrepreneurship, and servant leadership.

## Technical Details

This site is built using **Jekyll** and is based on a customized version of the *Feeling Responsive* theme.

* **Framework:** Foundation
* **Hosting:** GitHub Pages
* **Development:** Managed via Google Project IDX
